## Open vs Restricted Rule

Open character files answer:

**What does the reader and crew need to know now?**

Restricted character files answer:

**What hidden truth should shape behavior before the story reveals it?**

Final navigation principle:

**Open canon shows what a character does. Restricted canon explains what systems tried to do to them.**
