## 1. Red Umbrielor Identity Update

### Candidate Canon

Red Umbrielor's corrected naming stack is:

```text
MMS Quadtrack 580 = tractor size / class
Red Umbrielor = vehicle name
289 = call number
```

### Canon Reconciliation Needed

Current canon language in existing files may say:

```text
MMS Quadtrack 289, "Red Umbrielor"
```

That should be reviewed and likely corrected to avoid treating `289` as the model.

### Proposed Canon Wording

Use:

> **Red Umbrielor, an MMS Quadtrack 580 Traverse tractor, call number 289.**

Short forms:

- Red Umbrielor
- Red Umbrielor 289
- Quadtrack 580
- call 289

Avoid using `MMS Quadtrack 289` as the formal model/class line after canon reconciliation.

## 2. Quadtrack Model Status

### Candidate Canon

- Smaller MMS Quadtrack models exist.
- Smaller models are retired from active Traverse service.
- Red Umbrielor's active Traverse class/size is **MMS Quadtrack 580**.

### Visual Meaning

The 580 should visually read as the active heavy Traverse tractor class: large, articulated, four-track, heavily modified for crystalline-regolith Traverse work.

## 3. Red Umbrielor Exterior Markings

### Candidate Canon

Red Umbrielor's marking placement:

- `289` is stenciled only on the side of the engine compartment.
- `Red Umbrielor` is stenciled above the rear tracks on the side of the saddle tanks.

### Design Meaning

- `289` = practical call number.
- `Red Umbrielor` = vehicle name / machine identity.

Do not scatter the call number everywhere unless future canon deliberately expands maintenance markings.

## 4. Red Umbrielor Blade

### Candidate Canon

Red Umbrielor always has a front blade.

Blade details:

- multidirectional
- open, slightly box-style blade
- replaceable serrated blade edge

### Function

The blade supports Traverse work across crystalline regolith:

- route shaping
- crust breaking
- regolith pushing
- pressure-ridge testing
- growth-roller negotiation
- obstruction clearing

The blade should not read as decorative or military. It is serious work equipment.

## 9. MD-8 Fuel Handling Hardware

### Candidate Canon

Field fueling hardware consists of:

- pump and hoses lashed to a small sled
- sled and pump stored on the Magic Carpet when not in use

Fuel handling has:

- no warning labels
- black fuel bladders

## 10. MITE II Convoy Structure

### Candidate Canon

MITE II is a three-machine working Traverse convoy:

```text
1. 409
   NCI-modded crystal groomer
   lead sensor / terrain-reading tractor
   Ildi drives
   Forest reads terrain from rear cab
   pulls nothing

2. 289 Red Umbrielor
   MMS Quadtrack 580
   command / life-support tractor
   pulls Comms Mod, Solar/Berthing Mod, and fuel Tool train

3. 287 Discovery
   used MT-style crawler tractor
   Rudy's repair / sustainment tractor
   verbal nickname: Tight Tracks
   pulls Refrigerator/Storage Mod, Shed, and fuel Tool train
```

### Design Meaning

- 409 = eyes and nerves.
- Red Umbrielor = command spine and living body.
- 287 Discovery = hands, tools, repair, survival fallback.

## 11. 409 NCI-Modded Crystal Groomer

### Candidate Canon

409 is an **NCI-modded crystal groomer**.

Real-world design logic may come from ski-resort snow shaper / groomer vehicles, but in-world it is a crystalline-regolith Traverse machine.

### Naming

```text
409 = tractor number / call number
No name
No nickname
```

409 remains nameless.

### Crew Association

- Ildi Vey drives 409.
- Kavo "Forest" Rill rides in the rear cab.
- Forest reads terrain whenever the Traverse is in motion.
- 409 pulls nothing.

### GPR Boom System

409 has a front blade that mounts the GPR boom system:

- 20-foot boom mounted to the blade
- GPR unit on the leading end of the boom
- leading end of boom supported by electric winch cable
- winch cable passes through pulley on 8-foot mast mounted on the blade

This is a defining silhouette feature.

## 12. 287 Discovery / Rudy's Tractor

### Candidate Canon

Rudy Calder drives tractor 287.

287 is a used MT-style crawler tractor purchased by NCI.

It has:

- front winch
- rear knuckle-boom crane
- no visible nickname stencil

### Discovery Decal

The original owner decaled **DISCOVERY** across the top of the windshield.

NCI kept the decal or never bothered to remove it.

`Discovery` is inherited used-machine residue, not necessarily an official NCI name.

### Tight Tracks Nickname

Rudy's tractor is verbally nicknamed:

```text
Tight Tracks
```

Rules:

- verbal nickname only
- no visible `Tight Tracks` stencil

### Rudy Pull Chain

Rudy's tractor pulls:

```text
287 Discovery
→ Refrigerator/Storage Mod
→ Shed
→ Tool with two Magic Carpets carrying four black fuel bladders total
```

## 13. Mod Design Standard

### Candidate Canon

MITE II Mods are modified conex / seavan-style containers.

They ride on platforms attached to:

- four unpowered straight tracks

Important constraints:

- straight tracks
- not hi-tracks
- unpowered
- pulled by tractors
- platform-mounted

### Shared Visual Logic

Mods should read as:

- modified field containers
- practical
- boxy
- repaired
- insulated
- field-modified
- adapted for crystalline regolith

They should not read as sleek sci-fi pods or polished mobile homes.

## 15. Tools, Magic Carpets, and Fuel Bladder Layout

### Candidate Canon

Tools are known simply as:

```text
Tools
```

Example field speech:

- "Go get a Tool."
- "Parts came in for Tool 12."

### Tool Numbering

Tool numbers:

- are not visibly displayed in normal panels
- exist as small engraving on the tongue

### Tool Function

A Tool is a towable spreader bar used to attach two Magic Carpets side by side.

### Four-Bladder Arrangement

Four bladders ride as two front-to-back pairs on two side-by-side Magic Carpets:

```text
Tool / spreader bar
├─ Magic Carpet A
│  ├─ black fuel bladder 1
│  └─ black fuel bladder 2
└─ Magic Carpet B
   ├─ black fuel bladder 3
   └─ black fuel bladder 4
```

This layout applies to both fuel Tool trains currently assigned to MITE II.
