# Red Umbrielor Systems

> Canon reference for Red Umbrielor's onboard diagnostic language, fuel terminology, and common field-system distinctions.

**Canon Status:** MAIN CANON, EXPANDABLE
**Canon Weight:** Vehicle / Equipment / Procedure
**Scope:** Red Umbrielor, MITE II drafting, Traverse tractor system language
**Source of Truth:** Yes for MADS / MD-8 distinction

---

## Core Canon

**MADS** is Red Umbrielor's onboard diagnostic / alert system.

The name is simply:

**MADS**

Do not expand it into an acronym unless later canon deliberately does so.

MADS may:

- chime
- complain
- issue alerts
- read diagnostics
- report cab, engine, fuel, thermal, defrost, coupling, and route-system irregularities
- sound more judgmental than its designers probably intended

MADS is not a fully sentient AI.

It is an old industrial diagnostic system with enough voice, timing, and field history that crews treat it like part of Red Umbrielor's personality.

Canon line:

**MADS does not have feelings. Red Umbrielor's crews argue this point without evidence.**

---

## MD-8 Distinction

**MD-8** is not Red Umbrielor's computer.

**MD-8** stands for:

**Meridian Distillate 8**

MD-8 is fuel.

Use MD-8 for:

- fuel handling
- fuel planning
- fuel caches
- burn rate
- fuel smell
- fuel quality
- old-engine temperament
- Traverse logistics

Do not use MD-8 as the name of a diagnostic system, onboard computer, alert voice, AI, or control network.

Clean distinction:

| Term | Canon Function |
|---|---|
| **MADS** | Red Umbrielor's onboard diagnostic / alert system |
| **MD-8** | Meridian Distillate 8 fuel |

## Drafting Use

Correct:

> MADS chimed from the overhead diagnostic rail.

Correct:

> Red Umbrielor burned MD-8, Meridian Distillate 8, hard enough to make the cab smell like hot mineral oil and bad decisions.

Incorrect:

> The MD-8 system complained from the diagnostic rail.

Corrected version:

> MADS complained from the diagnostic rail while Red Umbrielor's MD-8 feed warmed through the line.

## Personality Boundary

MADS should be useful for field texture and machine character, but it should not become a comic sidekick.

MADS can feel:

- old
- dry
- stubborn
- procedural
- irritating
- oddly timed
- trusted despite itself

MADS should not feel:

- cute
- magical
- fully conscious
- modern consumer-tech slick
- secretly omniscient

MADS is a tool. The crew's relationship with it creates the personality.

---

## Relationship to Red Umbrielor

Red Umbrielor is the tractor.

MADS is one of the systems through which Red Umbrielor's problems become legible.

MADS should not replace Tenet's expertise.

Tenet can hear, feel, and read Red Umbrielor beyond what MADS reports.

Rudy Calder can often tell when MADS is technically correct but practically missing the point.

Possible Rudy logic:

**MADS tells you what failed. A mechanic tells you what lied before it failed.**
