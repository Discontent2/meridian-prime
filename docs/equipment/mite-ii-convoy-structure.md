## 409: Lead Sensor Tractor

**409** is the lead sensor tractor for MITE II.

It is a compact, NCI-modded crystal groomer adapted for Meridian Prime crystalline-regolith Traverse work.

409 is not a snowcat in-world.

409 is not a hauler.

409 is not a cargo tractor.

409 is the forward route-reading platform that puts sensors into the land before Red Umbrielor commits its mass.

### Naming and Numbering

```text
409 = tractor number / call number
No name
No nickname
```

409 should remain nameless and functional.

This makes it feel colder and less sentimental than Red Umbrielor or Rudy's used tractor.

### Crew Association

- **Ildi Vey** drives 409.
- **Kavo "Forest" Rill** rides in 409's rear cab / terrain-read station.
- Forest reads terrain whenever the Traverse is in motion.
- 409 pulls nothing.

### Story Function

409 gives MITE II visible terrain intelligence.

It should show that the route is read before Red Umbrielor and the heavier train commit.

This shifts Chapter 2 and Chapter 3 choreography:

- Ildi detects signal / system wrongness from 409's front driving and systems position.
- Forest reads terrain from 409's rear cab / terrain station.
- Red Umbrielor follows as command / life-support tractor.
- Discovery follows as Rudy's repair / sustainment tractor.

### GPR / Sensor Boom

409's defining silhouette feature is a front blade-mounted GPR boom system.

Planning-canon components:

- front blade as structural mount
- 20-foot GPR boom mounted to the blade
- GPR unit at the leading end of the boom
- 8-foot mast mounted on the blade
- pulley at or near the mast top
- electric winch cable supporting the boom tip / leading end
- protected cable routing
- rugged field-engineered look

The boom should feel competent, practical, and NCI-modified.

Avoid:

- magical scanner wand
- sleek sci-fi lance
- cargo crane silhouette
- polar snowcat identity

Travel Mode is the route configuration before Traverse Town is formed.

```text
409
pulls nothing

289 Red Umbrielor
→ Comms/Kitchen Mod

→ Tool

→ four black MD-8 bladders

287 Discovery
→ Refrigerator/Storage Mod

```

Travel Mode rules:

- 409 leads as the sensor tractor.
- 409 pulls nothing.
- Red Umbrielor follows as command / life-support tractor.
- Discovery follows as repair / storage / sustainment tractor.
- Mods remain in fixed connection order.
- Every segment uses tongue / drawbar logic unless later canon deliberately revises the mechanical language.
- Bladders remain attached at the back of each train.
- MD-8 is Meridian Distillate 8 fuel.

## Traverse Town Note

Traverse Town is the end-of-shift parked layout of MITE II.

In Traverse Town:

- the Mod trains park into two long parallel rows
- a central tractor alley forms between them
- 409 may park near the service / tractor side or slightly forward of the rows
- 409 remains separate because it pulls nothing
- Red Umbrielor anchors the life-support row
- Discovery anchors the repair / storage row

Exact Traverse Town choreography remains expandable, but the two-row working-convoy identity should be preserved.

## Chapter Choreography Implications

### Chapter 2

Chapter 2 should no longer imply that Red Umbrielor is the physical nose of the entire convoy.

Better choreography:

- 409 crawls ahead, reading the ground and signal environment.
- Ildi drives 409 from the forward systems position.
- Forest rides the rear terrain-read station.
- Red Umbrielor follows as Tenet's command / life-support tractor.
- Discovery follows as Rudy's repair / sustainment rig.

### Chapter 3

Chapter 3 should treat 409 as the front sensory machine.

The proposed extension corridor should be tested first through:

- 409's GPR boom
- Ildi's signal / infrastructure monitoring
- Forest's terrain read
- Red Umbrielor's command systems
- Discovery's machine-truth load audit

## Guardrails

- Do not make 409 sentimental.
- Do not give 409 a nickname unless later canon deliberately changes this.

- Do not collapse all crew functions into one vehicle.
- Do not import polar setting identity from groomer / snowcat reference.
