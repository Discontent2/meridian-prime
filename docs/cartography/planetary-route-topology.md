# Planetary Route Topology

> Canon route-adjacency skeleton for Meridian Prime's major regions, seasonal access logic, jurisdiction thresholds, and protected mystery boundaries.

**Canon Status:** MAIN CANON, EXPANDABLE / NON-FINAL TOPOLOGY

## Purpose

Give Meridian Prime a route-adjacency skeleton without locking exact final geography.

Core question:

**What connects to what, who controls the crossing, and what happens when the season changes?**

Meridian Prime currently has strong locations, factions, route systems, and seasonal ecology. This file prevents accidental shrinkage by defining how major spaces relate at a topology level.

This is not a final visual map.

This is not a complete atlas.

This is the skeleton that keeps the world from becoming a pile of cool places stacked in the same drawer.

Core canon principle:

**Meridian Prime is not a collection of cool places. It is a seasonal threshold world.**

Guiding principle:

**The official map is where authority says the road should be. The route topology is where survival says the world actually connects.**

---

## How To Use This File

Use this file to answer:

- which major regions connect to which
- which routes are direct, indirect, seasonal, sacred, corporate, outlaw, or protected mystery
- which factions control or contest each threshold
- which routes open, close, transform, or become dangerous by season
- which places are near Book One versus merely on the same planet
- which future files need adjacency, seasonal, and jurisdiction review

This file should be treated as expandable.

Do not use it to over-map sacred, hidden, or protected-mystery locations.

Canon line:

**Meridian Prime should feel vast because routes have cost, not because maps are vague.**

## Seasonal Route Matrix

Status labels:

- **Open**: passable under normal regional conditions.
- **Restricted**: passable only by permit, local authority, special vehicle, guide, ritual permission, or route proof.
- **Dangerous**: technically passable but high-risk.
- **Transformed**: route changes mode, such as road to water, dry channel to pirate archipelago, threshold road to sacred dock.
- **Closed**: effectively inaccessible under ordinary operations.
- **Contested**: access disputed or actively challenged.
- **Unknown / Protected**: do not define yet due protected mystery or incomplete canon.

This matrix is expandable and explicitly non-final.

## Pattern Repair Principles

### Planet as Port Network

Meridian Prime functions as a system of ports, thresholds, harbors, gates, customs regimes, sacred crossings, illegal docks, and route permissions.

Use this as structural logic, not literal maritime wallpaper.

Core line:

**Every major power controls a crossing. Every crossing creates politics.**

### Routes as Seasonal Circulatory System

Routes behave like arteries that open, clot, drown, freeze, reroute, or go septic.

Core line:

**The planet does not have roads. It has seasonal circulation.**

### Factions as Gatekeepers

Every major faction should be understood partly by what it controls access to.

Examples:

- Prime Ops controls official departure and survival logistics.
- World Works controls contracts, permissions, surveys, and infrastructure claims.
- Hydropolis controls legitimate Àæonos visitation.
- Craton authorities control gates and threshold law.
- Free Scabs control shadow movement through abandoned or misread routes.
- Bought Nation controls memory, labor, and settlement legitimacy but not enough gates.
- NCI controls field knowledge and practical machine-route truth.

Core line:

**A faction without a gate is either dependent, hidden, or becoming dangerous.**

### Maps as Contested Law

Maps are not neutral.

Different map traditions should include:

- official maps
- route maps
- seasonal maps
- sacred maps
- black-market maps
- ecological maps
- witness maps
- liability-shadow maps

Core line:

**A map is a legal argument wearing geography's coat.**

### Ecology as Infrastructure

Animals, plants, fungi, food systems, false shelters, and seasonal hazards function as infrastructure because they change travel, law, settlement, risk, and survival.

Core line:

**Ecology is infrastructure that eats, blooms, moves, rots, remembers, or lies.**

### Black Markets as Shadow Logistics

Black markets move what official maps, laws, and institutions refuse or fail to carry.

Core line:

**A black market is the logistics system of needs official maps refuse to carry.**

## Book One Distance Discipline

This topology protects Book One by separating:

- locations directly relevant to the Tenet / MITE II novella arc
- locations that exist during Book One but do not explain it
- civilization spaces that matter geopolitically but are not on-page requirements
- protected mysteries that create motive without proof

Hydropolis, the Anticlinal Craton, Eschar Slough, and Blackstep Province may exist during Book One without becoming adjacent to every Book One event.

The Dry Port hypothesis may motivate World Works without being confirmed.

Canon line:

**A place can be important without being close. A mystery can exert pressure without becoming an answer.**
